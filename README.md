## NB2020REACT

#### 描述
根据公司同事需求进行开发一个企业级管理平台

#### 软件架构
React + React Hook + React Router + Axios + Ant Design

#### 安装
1. yarn install (安装依赖包)
2. yarn start （本地启动服务）
3. yarn build   (打包)
4. yarn global add serve (本地serve 启动静态服务)
5. serve -s build (启动服务)

#### 包结构
1. src/config   接口配置文件
2. src/pages    页面文件
3. src/static  静态资源文件(css/静态图片)
4. index.js     入口文件
